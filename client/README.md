# client

## Cd to client directory
```
cd client
```

## Install node
Use node v14, preferably installed via nvm https://github.com/nvm-sh/nvm
https://www.freecodecamp.org/news/node-version-manager-nvm-install-guide/

install node v14 via nvm
```
nvm install 14
nvm use 14
```

```
npm install
```
